# VUI

> ui component for V2.0

## Included
 - **Picker**
 - **addressPicker**
 - **datatime**
 - **dialog**
 - **selectPicker**
 - **switper**
 - **switch**
 - **countup**

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
